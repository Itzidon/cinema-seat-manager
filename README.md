# Cinema Seat Manager

Proyecto desarrollado en TypeScript para gestionar una sala de cine mediante una matriz bidimensional.

## Funciones

- Crear una sala de cine de 8 filas y 10 columnas
- Mostrar el estado de los asientos
- Reservar asientos
- Validar asientos ocupados o inválidos
- Contar asientos disponibles y ocupados
- Buscar asientos contiguos

## Tecnologías

- TypeScript
- Node.js
